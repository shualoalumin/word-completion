## TOEFL iBT 점수 구조

### 전체 점수 체계

- **총점**: **0-120점** (4개 섹션 합산)
- **섹션별 점수**: 각 섹션 **0-30점**
- **섹션 구성**:
  1. **Reading** (읽기): 0-30점
  2. **Listening** (듣기): 0-30점
  3. **Speaking** (말하기): 0-30점
  4. **Writing** (쓰기): 0-30점

### 점수 등급

| 총점 범위 | 등급 | 설명 |
|----------|------|------|
| **100-120** | 우수 (Advanced) | 대부분의 대학 입학 기준 충족 |
| **80-99** | 양호 (Good) | 많은 대학 입학 기준 충족 |
| **60-79** | 보통 (Fair) | 일부 대학 입학 가능 |
| **0-59** | 부족 (Limited) | 추가 학습 필요 |

## 현재 구현 상태 기준 예측 점수

### 현재 상황

- ✅ **Reading 섹션만 구현됨** (Text Completion)
- ❌ Listening, Speaking, Writing 섹션 미구현

### 예측 점수 제공 시점

#### 1. 현재 (Reading만 구현)

**제공 가능한 점수**:
- Reading 섹션 예측 점수: **0-30점**
- 총점 범위 추정: Reading 점수 × 4 배율 (정확도 낮음)

**제한사항**:
- 4개 섹션 중 1개만 평가 가능
- 총점 예측의 신뢰도 낮음 (×4 배율은 근사치)

**권장 메시지**:
```
"현재 Reading 섹션만 평가 가능합니다.
예상 Reading 점수: 22/30
(전체 TOEFL 점수 예측은 4개 섹션 모두 구현 후 제공됩니다)"
```

#### 2. 4개 섹션 모두 구현 후

**제공 가능한 점수**:
- 각 섹션별 예측 점수: Reading, Listening, Writing, Speaking (각 0-30)
- 총점 예측: 0-120점 (정확도 높음)
- CEFR 레벨 추정: 정확한 레벨 판정 가능

**제공 방식**:
- 실시간 예측: 문제 풀이 후 즉시 업데이트
- 주간/월간 추세: 성장 곡선과 함께 표시
- 목표 대비 진행도: "목표 100점까지 15점 남았습니다"

## 점수 예측 알고리즘 (향후 구현)

### Reading 섹션 점수 계산 (현재 가능)

```typescript
interface ReadingScorePrediction {
  // 기본 점수 계산
  rawScore: number;           // 실제 문제 풀이 점수 (0~100%)
  
  // TOEFL 점수 환산
  toeflReadingScore: number;  // 0~30점
  
  // 신뢰도
  confidence: number;         // 0~1 (신뢰도)
  
  // 세부 분석
  breakdown: {
    vocabulary: number;       // 어휘 문제 점수
    comprehension: number;    // 이해 문제 점수
    inference: number;        // 추론 문제 점수
  };
}

function predictReadingScore(
  avgAccuracy: number,        // 평균 정답률 (0~1)
  exercisesCompleted: number, // 완료한 문제 수
  difficultyDistribution: {   // 난이도별 분포
    easy: number;
    intermediate: number;
    hard: number;
  }
): ReadingScorePrediction {
  // 1. 기본 점수 계산 (가중 평균)
  const weightedScore = (
    difficultyDistribution.easy * avgAccuracy * 0.7 +
    difficultyDistribution.intermediate * avgAccuracy * 0.9 +
    difficultyDistribution.hard * avgAccuracy * 1.1
  ) / (
    difficultyDistribution.easy +
    difficultyDistribution.intermediate +
    difficultyDistribution.hard
  );
  
  // 2. TOEFL 점수 환산 (0~30)
  const toeflScore = Math.round(weightedScore * 30);
  
  // 3. 신뢰도 계산 (문제 수에 따라)
  const confidence = Math.min(1, exercisesCompleted / 50); // 50문제 이상 시 최대 신뢰도
  
  return {
    rawScore: weightedScore * 100,
    toeflReadingScore: toeflScore,
    confidence,
    breakdown: {
      vocabulary: calculateVocabularyScore(),
      comprehension: calculateComprehensionScore(),
      inference: calculateInferenceScore()
    }
  };
}
```

### 총점 예측 (4개 섹션 모두 구현 후)

```typescript
function predictTotalScore(
  reading: number,      // 0~30
  listening: number,    // 0~30
  writing: number,      // 0~30
  speaking: number      // 0~30
): {
  totalScore: number;   // 0~120
  cefrLevel: string;    // 'A1' ~ 'C2'
  confidence: number;   // 0~1
} {
  const totalScore = reading + listening + writing + speaking;
  
  // CEFR 레벨 판정
  let cefrLevel: string;
  if (totalScore >= 114) cefrLevel = 'C2';
  else if (totalScore >= 106) cefrLevel = 'C1 (5.5)';
  else if (totalScore >= 95) cefrLevel = 'C1 (5)';
  else if (totalScore >= 86) cefrLevel = 'B2 (4.5)';
  else if (totalScore >= 72) cefrLevel = 'B2 (4)';
  else if (totalScore >= 58) cefrLevel = 'B1 (3.5)';
  else if (totalScore >= 44) cefrLevel = 'B1 (3)';
  else if (totalScore >= 35) cefrLevel = 'A2 (2.5)';
  else if (totalScore >= 24) cefrLevel = 'A2 (2)';
  else if (totalScore >= 12) cefrLevel = 'A1 (1.5)';
  else cefrLevel = 'A1 (1)';
  
  return {
    totalScore,
    cefrLevel,
    confidence: calculateConfidence(reading, listening, writing, speaking)
  };
}
```
